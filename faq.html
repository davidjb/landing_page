Frequently Asked Questions

Does TellForm have Conditional Logic/Logic Jump?

No, currently TellForm does not have Logic Jump or Conditional Logic for it's fields (although this is a feature we are looking to add in the coming months)

Does TellForm have Upload Fields?

No, currently TellForm does not have upload fields. However, we currently are working on a file upload feature that will allow users to link their AWS-S3 accounts.

Can I use TellForm without setting it up on my own server?

Yes you can, by signup up to our hosted service at http://admin.tellform.com/#!/signup.

How do I setup TellForm on my own server

You can follow our quickstart tutorial on our README here. If you have any questions or problems feel free to make a GitHub issue!

I want to propose a new feature for TellForm. Where can I do this?

If you want to propose a new feature for TellForm you can make a new issue on our GitHub repository and we'll be sure to get back to you as soon as possible.
